# Gerenciados de Miniaturas Hot Wheels

# Variáveis de ambiente necessárias
```
URL_CONNECTION : A URL de conexão JDBC com um banco PostgreSQL
USER_NAME      : O usuário do banco de dados
PASSWORD       : A senha do banco de dados
ADMIN_PASSWORD : A senha que será usada na criação do usuário de administração. Você vai logar no sistema com o usuário 'admin' e esta senha.
```
